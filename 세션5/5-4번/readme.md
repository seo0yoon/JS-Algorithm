연속 부분수열 2
N개의 수로 이루어진 수열이 주어집니다.
이 수열에서 연속부분수열의 합이 특정숫자 M이하가 되는 경우가 몇 번 있는지 구하는 프로그램을 작성하세요.
만약 N=5, M=5이고 수열이 다음과 같다면
1 3 1 2 3
합이 5이하가 되는 연속부분수열은 {1}, {3}, {1}, {2}, {3}, {1, 3}, {3, 1}, {1, 2}, {2, 3},
{1, 3, 1}로 총 10가지입니다.
▣ 입력설명
첫째 줄에 N(1≤N≤100,000), M(1≤M≤100,000,000)이 주어진다.
수열의 원소값은 1,000을 넘지 않는 자연수이다.
▣ 출력설명
첫째 줄에 경우의 수를 출력한다.
▣ 입력예제 1
5 5
1 3 1 2 3
▣ 출력예제 1
10

1. 변수를 생성한다.
2. 첫번째 반복문은 1, 3, 1, 2, 3이 5이하인지 확인한 것,
   (여기까지 카운터가 5개)
3. 반복문을 하나 더 사용해서 이번엔 j부터 돌아야하니 [3,1][1,2][2,3] 까지 5이하면 카운터 1을 올려준다. (여기까지 카운터 총 8개)
4. 그다음 while문사용.
   => [1],[1,3],[1,3,1],[1,3,1,2],[1,3,1,2,3]
